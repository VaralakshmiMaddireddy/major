# 📰 Fake News Detection using Transformer Models  

## 📌 Overview  
This project focuses on detecting fake news using advanced transformer-based models such as **BERT** and **DistilBERT**. The system classifies news articles as *real* or *fake* by leveraging deep learning techniques in **Natural Language Processing (NLP)**.

---

## 🎯 Objectives  
- Build a robust fake news classification system  
- Compare performance of BERT and DistilBERT  
- Achieve high accuracy using transformer architectures  
- Analyze results using confusion matrices and evaluation metrics  

---

## 🧠 Research Domain  
- Deep Learning  
- Natural Language Processing (NLP)  
- Transformer-based Text Classification  

---

## 🌍 Application Area  
- Information Security  
- News and Media Systems  
- Misinformation Detection  

---

## 🛠️ Technologies Used  
- Python  
- TensorFlow / PyTorch  
- Hugging Face Transformers  
- Scikit-learn  
- Matplotlib  

---

## ⚙️ Models Used  
- **BERT** (Bidirectional Encoder Representations from Transformers)  
- **DistilBERT** (Lightweight version of BERT)  

---

## 📊 Evaluation Metrics  
- Accuracy  
- Precision  
- Recall  
- F1-Score  
- Confusion Matrix  

---

## 📈 Results  
- **BERT Accuracy:** ~97.5%  
- **DistilBERT Accuracy:** ~97.2%  

> DistilBERT achieves comparable performance with reduced computational cost.

---

## 🚀 Conclusion  
This project demonstrates the effectiveness of transformer-based models in fake news detection. While BERT provides slightly higher accuracy, DistilBERT offers a more efficient alternative with faster performance and lower resource consumption.
